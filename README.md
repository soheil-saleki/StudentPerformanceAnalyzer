# Student Performance Analyzer

A Python project to analyze the performance of a class of students.  
It calculates individual averages, class average, identifies top and weakest students, and determines pass/fail status.

## Features

- Calculate average score for each student
- Calculate overall class average
- Identify top and weakest students
- Determine pass/fail status based on average score
- Print a clean, readable report in the terminal

## Files

- `StudentPerformanceAnalyzer.py` — main Python script
- `student.txt` — sample student scores

## Usage

1. Ensure Python is installed on your computer.
2. Place `student.txt` in the same folder as `StudentPerformanceAnalyzer.py`.
3. Run the script:

```bash
python StudentPerformanceAnalyzer.py
